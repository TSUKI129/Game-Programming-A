# ゲームプログラミング演習・実習A
## 担当教員： 向井智彦 (Email: )

## 授業方針・テーマ
　インタラクティブグラフィックスシステムの代表例であるビデオゲームを題材として、システムの設計からプログラミング、テストに至る一連のソフトウェアデザインプロセスを修得する。


## 到達目標: 
- C/C++言語によるローカルアプリケーションプログラミングの基礎の習得
- C/C++言語によるローカルアプリにおける2次元グラフィック、3次元グラフィックのゲームの構築
